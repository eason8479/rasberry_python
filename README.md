# rasberry_python

## 檔案用途
- manual.py：測試手動模式
- pi_control.py：測試pi的pin腳
- project_for_debug.py：在電腦上測試用的程式
- project_for_pi.py：需要燒進pi的程式
- random_1.py：隨機數測試
- ust_test.py：usb

## pin腳用途
 \ | GPIO2 | GPIO3
---| --- | ---
on | 開門 | 開夾爪
off| 關門 | 關夾爪
